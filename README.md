# toolNav 本地开发启动说明

## 启动本地静态服务器

本项目已提供最简单的 Python 静态服务器启动脚本，无需任何第三方依赖。

### 步骤
1. 确保已安装 Python 3（macOS 默认自带）。
2. 在项目根目录下运行：

```
python3 server.py
```

3. 打开浏览器访问：http://localhost:8000

即可访问所有前端页面。

---

### 自定义端口
如需自定义端口（如 8088），可使用如下命令：

```
python3 server.py --port 8088
```

如需更换端口，也可以通过上述命令行参数实现，无需修改代码。
